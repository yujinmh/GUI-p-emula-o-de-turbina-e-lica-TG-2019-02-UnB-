Tema: Desenvolvimento de Interface de Comunicação Baseada no Protocolo Modbus
para Conexão de um Computador (PC) a um Inversor de Frequência

Orientador: Lélio Ribeiro Soares Júnior

O código utiliza Python versão 3.7.0 para rodar. É possível executá-lo em plataformas Windows e Linux, utilizando o 
comando 'py tg_interface.py'. É necessário instalar algumas bibliotecas para executar o código, tais como: serial e
minimalmodbus e tkinter (sendo o último um pacote padrão de interface do Python).

A interface abrirá em uma nova janela após a execução do comando citado anteriormente. Nela, é possível criar
configurações de velocidades e durações de diferentes tipos de ventos préprogramados, salvar a configuração criada
pelo usuário em um arquivo .txt (no mesmo diretório em que se encontra o arquivo .py do código) e/ou carregar um .txt
criado previamente com as configurações desejadas.
